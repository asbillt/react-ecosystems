# Todo List App

A React Redux Application built to keep track of incomplete and completed todo items. Todo items can be created, removed or marked as completed.

# Tech Used

**Built With:**

- React.js
- Redux
- HTML5
- CSS3
- Babel
- Webpack
- Visual Studio Code
- Git
- Github
- NPM
